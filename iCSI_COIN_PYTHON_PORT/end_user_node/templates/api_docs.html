üîê Authentication

Login

POST
/api/auth/login

Authenticates a session and returns a cookie.

# Login and save cookie
curl -c cookies.txt -X POST http://localhost:5000/api/auth/login \
     -H "Content-Type: application/json" \
     -d '{"username": "admin", "password": "securepassword"}'

Setup

POST
/api/auth/setup

Sets admin credentials (first run only).

üí∞ Wallet

List Wallets

AUTH GET
/api/wallet/list

Returns all wallets and balances.

curl -b cookies.txt http://localhost:5000/api/wallet/list

Create Wallet

AUTH POST
/api/wallet/create 
curl -b cookies.txt -X POST http://localhost:5000/api/wallet/create \
     -H "Content-Type: application/json" \
     -d '{"name": "Savings"}'

Send Coins

AUTH POST
/api/wallet/send 
curl -b cookies.txt -X POST http://localhost:5000/api/wallet/send \
     -H "Content-Type: application/json" \
     -d '{
           "from": "1SourceAddr...",
           "to": "1DestAddr...", 
           "amount": 50
         }'
